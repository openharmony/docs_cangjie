# ContextMenuController

Provides the capability to control menu closure.

> **Note:**
>
> To use the following APIs, first obtain a ContextMenuController instance via the [getContextMenuController()](./cj-apis-uicontext-uicontext.md#func-getcontextmenucontroller) method from [UIContext](./cj-apis-uicontext-uicontext.md#class-uicontext), then call the corresponding methods through this instance.
