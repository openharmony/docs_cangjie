# Using ArkTS in Cangjie Applications

When developing Cangjie applications, you can leverage ArkTS's rich library ecosystem by calling existing ArkTS libraries. Cangjie applications do not support adding ArkTS pages, but support adding ArkTS modules. ArkTS libraries are divided into NAPI (Node.js API) modules and third-party library modules. Currently, Cangjie supports calling NAPI modules but does not support third-party library modules. For details on how to call NAPI modules, refer to [Cangjie Calling ArkTS](./using_arkts_module.md).
