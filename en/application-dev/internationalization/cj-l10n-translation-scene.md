# Providing Translation Context

<!--Del-->
> **Note:**
>
> Currently in the beta phase.
<!--DelEnd-->

The same content may yield different translation results depending on the scenario and context. When providing interface strings for translation, complete contextual information should be given to translators to avoid translation deviations. Translation context information typically includes the following two types:

1. Annotations or remarks in string resource files, including: contextual scenarios, part of speech, key term definitions, maximum allowable text length for controls, variable meanings and their value ranges.
2. Interface screenshots.
